# วิดีโอสำหรับ Carousel

## ไฟล์วิดีโอที่ต้องการ

### วิดีโอหลัก
- `jazz-bar-preview.mp4` - วิดีโอแนะนำ Jazz Bar (รูปแบบ MP4)
- `jazz-bar-preview.webm` - วิดีโอแนะนำ Jazz Bar (รูปแบบ WebM สำหรับ browser ที่รองรับ)

### ภาพปกวิดีโอ
- `video-poster.jpg` - ภาพปกสำหรับวิดีโอ (แสดงก่อนเล่นวิดีโอ)

## ข้อกำหนดวิดีโอ

### ขนาดและคุณภาพ
- **ความละเอียด**: 720p (1280x720) หรือ 1080p (1920x1080)
- **อัตราส่วน**: 16:9 หรือ 4:3
- **ความยาว**: 30-60 วินาที (แนะนำ)
- **ขนาดไฟล์**: ไม่เกิน 10MB

### รูปแบบไฟล์
- **MP4**: H.264 codec, AAC audio
- **WebM**: VP9 codec, Vorbis audio
- **ความถี่**: 30fps หรือ 60fps

### เนื้อหาที่แนะนำ
- บรรยากาศภายใน Jazz Bar
- การแสดงดนตรีแจ๊สสด
- การเตรียมเครื่องดื่ม
- บรรยากาศการนั่งดื่ม
- ภาพรวมของร้าน

## วิธีการใส่วิดีโอ

1. เตรียมวิดีโอตามข้อกำหนดด้านบน
2. ใส่ไฟล์วิดีโอในโฟลเดอร์ `videos/`
3. ใส่ภาพปกในโฟลเดอร์ `images/`
4. ตรวจสอบว่าไฟล์ชื่อตรงกับที่ระบุใน HTML
5. ทดสอบการเล่นวิดีโอ

## ฟีเจอร์วิดีโอ

### การควบคุม
- คลิกที่ปุ่มเล่นเพื่อเริ่ม/หยุดวิดีโอ
- วิดีโอจะหยุดอัตโนมัติเมื่อเปลี่ยนไป slide อื่น
- รองรับการเล่นซ้ำเมื่อจบวิดีโอ

### การแสดงผล
- วิดีโอจะปรับขนาดอัตโนมัติให้พอดีกับ carousel
- มี overlay แสดงปุ่มเล่นและข้อความ
- รองรับการแสดงผลบนมือถือ

### Fallback
- หากวิดีโอไม่สามารถโหลดได้ จะแสดงข้อความแจ้งเตือน
- รองรับ browser ที่ไม่รองรับ video tag

## หมายเหตุ

- วิดีโอจะถูกโหลดเมื่อเข้าถึง slide ที่ 4
- ควรใช้ไฟล์ที่มีขนาดเหมาะสมเพื่อการโหลดที่เร็ว
- แนะนำให้ใช้ CDN หรือ hosting service สำหรับไฟล์วิดีโอขนาดใหญ่
